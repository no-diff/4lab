# Lab 4 - Streamlit Data Analysis

Інтерактивний застосунок для аналізу кліматичних індексів (VCI, TCI, VHI) по областях України.

## Функціонал

* вибір індексу (VCI, TCI, VHI)
* фільтрація за областю
* вибір діапазону років і тижнів
* сортування значень
* таблиця з даними
* графік часових рядів
* порівняння кількох областей

## Структура

* `lab_4_1.ipynb` — основний код застосунку
* `household_power_consumption.csv` — дані (якщо використовується)
* `../lab2/data/` — джерело CSV файлів з даними

## Встановлення

```bash
pip install streamlit pandas matplotlib
```

## Запуск

```bash
streamlit run lab_4_1.ipynb
```

Якщо виникає помилка, збережіть файл як `.py` і запустіть:

```bash
streamlit run app.py
```

## Примітки

* дані автоматично очищуються від некоректних значень
* HTML-теги видаляються перед обробкою
* використовується кешування для пришвидшення роботи
* дані об'єднуються по всіх областях в один DataFrame
